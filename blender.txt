---------- OLD BLENDER ----------
Q1: RuntimeError: Armature 'proportions' not in edit mode, cannot add an editbone

A1: When you're importing the .smd, make sure to first set 'Bone Append Mode' to 'Make New Armature', then import the following model pieces with 'Append to Target'.

Q2: When I import my model, the skeleton is much smaller than the model's mesh.

A2: In new Blender, apply all transformations (with Ctrl+A) on everything in the scene. Then, re-export the model.
